Scramble Version 2.0

The most important aspect of Scratch are the blocks, divided into eight categories. Blocks are like procedures, that execute some code. The problem is that there is a limited amount of blocks that come with the package. More importantly, making them is not straightforward at all! This is where Scramble comes in. Scramble allows you to create your own blocks in an interactive form-based environment. Scramble was created by Hardmath123. The best part is that it runs right in your web browser, so you don't need to install anything. Enjoy!
Hardmath123

Step I: The Blueprint

Firstly, we'll describe our block. This step will create the coding that renders your block. This is called the blockspec.

Type a 3 to 5 symbol name for your block. It needs to use only the 26 letters, or their capitalized versions.

Step II: Code it!

Add temporary variables

Click the add button to add temporary variables. You currently have: temporary variables.
To access the nth temporary variable in the script, use tn, i.e. the first variable is t1, the second is t2, and so on. To access the values added in the block's inserter(s), use i1, i2, etc. in order of appearance in the blockspec.

The last bit. . .

The hard part is to actually code it - write down what the block is going to do, and how. Click a button to get that block. Fill out all the inputs, then click add to add it to the script.
